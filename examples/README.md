# DataEng Toolbox Examples

This directory contains example scripts demonstrating how to use the dataeng_toolbox package.

## Available Examples

### basic_usage.py
Demonstrates the basic functionality of the Core class, including:
- Creating a Core instance
- Using the hello_world() method
- Retrieving toolbox information

## Running Examples

To run any example, navigate to the root directory of the package and run:

```bash
python examples/basic_usage.py
```

Or if you have the package installed:

```bash
cd examples
python basic_usage.py
```

## Adding New Examples

When adding new examples:
1. Create a new Python file in this directory
2. Include proper documentation and comments
3. Update this README with a description of the new example
4. Follow the same structure as existing examples
